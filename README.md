# 🛡️ DUNGEON HUB | Tactical Control

**DUNGEON HUB** es una plataforma de coordinación en tiempo real diseñada para equipos de incursión en MMORPGs (Solo Leveling, One Piece, etc.). Permite la creación de salas privadas con encriptación por PIN para una comunicación táctica sin interferencias.

## 🚀 Características
* **Lobby Global:** Visualización de frecuencias (salas) activas en tiempo real.
* **Salas Privadas:** Conexión segura mediante ID de sala y PIN de 4 dígitos.
* **Alertas de Audio:** Notificaciones sonoras instantáneas para niveles Easy, Medium y Hard.
* **Arquitectura Serverless:** Ejecutado en el Edge con Cloudflare Workers para latencia mínima.

## 🛠️ Stack Tecnológico
* **Frontend:** HTML5, Tailwind CSS, Orbitron Typography.
* **Real-time:** Pusher Channels API.
* **Backend:** Cloudflare Workers (Runtime de JavaScript).
